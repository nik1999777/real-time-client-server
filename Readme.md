## Usage
### Installing dependencies
To install dependencies, run the command:
```sh
$ yarn
```

### Start client
To start the development client, run the command:
```sh
$ yarn run dev
```
```sh
$ yarn run build
```
```sh
$ yarn run preview
```

### Start server
To start the development server, run the command:
#### Start Long polling
```sh
$ yarn run start_1
```
#### Start Event sourcing
```sh
$ yarn run start_2
```
#### Start Websocket
```sh
$ yarn run start_3
```
